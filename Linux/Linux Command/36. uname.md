# Lệnh ‘uname’ nổi bật như một công cụ đa năng cung cấp các chi tiết quan trọng về máy Linux. Các thông tin về hệ thống bao gồm phiên bản kernel, hệ điều hành, kiến trúc phần cứng, ...

## Cấu trúc: 
```
uname [OPTIONs]
```
